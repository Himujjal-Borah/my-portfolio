# MyPortfolio
AI Portfolio Website with Chatbot & Automation

An advanced personal portfolio website with an integrated AI chatbot + automation backend (n8n).

🧠 Core Idea

This project is not just a portfolio — it’s a smart interactive system where:

Users can chat with AI about me

Send messages via contact form automation

Experience a modern animated UI

⚙️ Tech Stack

🎨 Frontend: HTML, CSS, JavaScript

⚙️ Backend: n8n (Webhook-based automation)

🤖 AI: Google Gemini

📩 Email: SMTP
